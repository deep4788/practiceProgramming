Simple Unix Shell
=================

A simple Unix shell interpreter (e.g., bash, csh etc.).

How to Build
------------
```sh
$ mkdir build bin lib
$ cd build
$ cmake ..
$ make
```

Usage
-----
```sh
$ cd bin
$ ./simpleShell
```
