Simple Unix Shell
=================

A simple Unix shell interpreter (e.g., bash, csh etc.).

How to Build
------------
```sh
$ mkdir build bin lib
$ cd build
$ cmake ..
$ make
```

Usage
-----
Running shell
```sh
$ cd bin
$ ./simpleShell
```
<br>
Running unit-tests
```sh
$ cd bin
$ ./test_log
```
